⛔ System-level MUST NOT Policy（Dispatcher 強制執行）

  • Sub-agents 編輯 state.md / progress_log.md / validation_log.md（唯一寫入者是 dispatcher）
  • Sub-agents 讀取同 batch 其他 peer 的 workspace / findings（寫面必須互斥）
  • Sub-agent 不輸出 ---agent-return--- YAML 區塊（缺則記 [BAD_RETURN]，視為崩潰重 spawn）
  • validator escalate_to_blocked:true 的任務被靜默重試（直接 AWAITING_HUMAN）
  • 同 batch 兩 executor 的 Touches 來源路徑重疊（規劃期 PR-013 設為 DAG 相依）
  • 對工單系統寫回（只讀 work item body + 驗收標準）
